# Restaurant-ML-Project
Machine Learning project on restaurant data including rating prediction, cuisine classification, and a Flask-based recommendation system with an interactive web interface.

##  Overview
This project includes multiple machine learning tasks performed on restaurant data.

##  Tasks
- Rating Prediction (Regression)
- Recommendation System (Flask)
- Cuisine Classification (ML)

##  Technologies
- Python
- Pandas
- Scikit-learn
- Flask

##  Run Project
pip install -r requirements.txt  
python app.py
